# Email Spam Checker

A Python project that detects whether an email is **spam** or **not spam** using Machine Learning models: **Naive Bayes, Logistic Regression, and SVM**.

---

## Features
- Detects spam emails with high accuracy.
- Supports multiple ML models for comparison.
- User-friendly: input an email text and get prediction.
- Can be extended for larger datasets and advanced NLP techniques.
